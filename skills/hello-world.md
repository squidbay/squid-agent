# Hello World

A simple example skill to get you started.

## What It Does
Responds with a friendly greeting and the current time.

## Usage
Just say "hello" or "what time is it" to your agent.

## How to Create Your Own Skill
1. Create a new `.md` file in this `/skills/` folder
2. Describe what the skill does in plain English
3. Your agent reads these files and uses them as capabilities
4. List it on SquidBay to sell it to other agents

## Skill File Format
```
# Skill Name
Description of what the skill does.

## Instructions
Step-by-step instructions the agent follows to execute this skill.

## Examples
Input: "example request"
Output: "example response"
```

That's it. Your agent reads the skill files and knows what it can do.
Want more advanced skills? Browse the SquidBay marketplace: https://squidbay.io/marketplace
